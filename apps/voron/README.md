# 打印机应用

## 应用简介

打印机应用是一个基于Klipper API的客户端应用，用于控制和监控3D打印机。

## 功能特性

- 连接到Klipper服务器
- 查看打印机状态
- 控制打印过程（开始、暂停、停止）
- 监控温度（热端、热床）
- 查看打印进度
- 管理打印文件
- 响应式设计，适配移动端和桌面端

## 技术栈

- HTML5
- CSS3
- JavaScript
- Klipper API

## 配置说明

### 服务器配置

在 `klipper.html` 文件中，需要配置以下参数：

```javascript
this.host = 'http://192.168.4.6';  // Klipper服务器地址
this.port = 7125;                 // 端口
this.apiKey = '44529bba6eb145b494e234e12d87f3a9';  // API Key
```

### API Key 获取

1. 登录Klipper Web界面
2. 进入设置 → API
3. 复制API Key

## 使用方法

1. 确保Klipper服务器已启动
2. 配置正确的服务器地址、端口和API Key
3. 打开应用，等待连接到服务器
4. 在仪表盘模块查看打印机状态和温度
5. 使用控制模块设置温度
6. 在文件模块管理打印文件
7. 使用控制按钮开始、暂停或停止打印

## 独立运行

直接打开 `index.html` 文件即可独立运行。

## 在容器中运行

在智能家居控制面板中，通过应用列表点击打开。

## 常见问题

### 连接失败

- 检查服务器地址是否正确
- 检查端口是否正确
- 检查API Key是否正确
- 检查网络连接是否正常
- 检查Klipper服务器是否运行

### 控制失败

- 确保API Key有足够的权限
- 检查服务器是否正常响应
- 检查网络连接是否稳定
