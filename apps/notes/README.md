# 笔记应用

## 应用简介

笔记应用是一个基于思源笔记API的客户端应用，用于管理和编辑笔记内容。

## 功能特性

- 连接到思源笔记服务器
- 查看笔记本和文档列表
- 查看和编辑笔记内容
- 支持Markdown格式
- 响应式设计，适配移动端和桌面端

## 技术栈

- HTML5
- CSS3
- JavaScript
- 思源笔记API

## 目录结构

```
apps/notes/
  index.html          # 应用入口文件
  assets/             # 资源文件
  js/                 # JavaScript文件
  css/                # 样式文件
  config/             # 应用配置
  README.md           # 应用说明文档
```

## 配置说明

### 服务器配置

在 `index.html` 文件中，需要配置以下参数：

```javascript
this.state = {
    serverUrl: 'http://192.168.4.13:6806',  // 思源笔记服务器地址
    apiToken: 'kynm8bx11zmhgdst',          // API Token
    // 其他配置...
};
```

### API Token 获取

1. 登录思源笔记
2. 进入设置 → 关于
3. 复制API Token

## 使用方法

1. 确保思源笔记服务器已启动
2. 配置正确的服务器地址和API Token
3. 打开应用，等待连接到服务器
4. 选择笔记本和文档查看内容
5. 点击编辑按钮修改内容
6. 点击保存按钮保存修改

## 独立运行

直接打开 `index.html` 文件即可独立运行。

## 在容器中运行

在智能家居控制面板中，通过应用列表点击打开。

## 常见问题

### 连接失败

- 检查服务器地址是否正确
- 检查API Token是否正确
- 检查网络连接是否正常
- 检查思源笔记服务器是否运行

### 编辑失败

- 确保有足够的权限
- 检查服务器是否正常响应
- 检查网络连接是否稳定
